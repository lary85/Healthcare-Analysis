# Healthcare-Treatment-Analysis-Report
This project presents an interactive Hospital Appointment Analysis Dashboard built using Power BI, designed to provide clear insights into patient visits, treatment trends, and revenue performance.

### Overview

This project showcases an interactive **Hospital Appointment Analysis Dashboard** built using **Microsoft Power BI**. It provides a comprehensive view of hospital operations, patient behavior, and revenue performance through intuitive visualizations and key performance indicators (KPIs).

The goal of this project is to demonstrate how healthcare data can be transformed into actionable insights to support better decision-making.

### Dashboard Preview

![Dashboard Screenshot](https://github.com/lary85/Healthcare-Analysis/blob/main/Hospital%20appointment%20%20analysis.png))

### Key Features

#### 1. High-Level Metrics

* Total Appointments: 5,000
* Total Patients: 1,824
* Total Doctors: 100
* Total Revenue: $1.75M
* Treatment Types: 6

#### 2. Patient Insights

* Appointment reasons (Checkup, Malaria, Diabetes Review, etc.)
* Gender distribution analysis

#### 3. Operational Trends

* Monthly appointment trends
* Visits categorized by treatment type:

  * Consultation
  * CT Scan
  * Therapy Session
  * Medication
  * Lab Test
  * X-Ray

#### 4. Revenue Analysis

* Monthly revenue trends
* Revenue generated by appointment reasons

#### 5. Interactive Filters

* Treatment Type
* Gender
* Doctor’s Specialization


### Tools & Technologies Used

* **Microsoft Excel / CSV** – Data source (sample dataset)
* * **Power Query** – Data cleaning and transformation
* **Power BI Desktop** – Data modeling and dashboard creation
* **DAX (Data Analysis Expressions)** – Calculated measures and KPIs
